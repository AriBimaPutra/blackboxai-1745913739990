# MyPortfolio

A personal portfolio website for showcasing the skills, projects, and contact details of a digital marketer and graphic designer, Ari Bima Putra.

## Project Overview

MyPortfolio is designed to highlight Ari Bima Putra's professional journey, skills, and portfolio items. The site features a responsive design that adapts to different screen sizes, making it accessible on both desktop and mobile devices. Key sections include an introduction, portfolio showcase, skills list, and contact information.

## Installation

To get started with MyPortfolio, you can clone the repository and open the `index.html` file in your preferred web browser:

```bash
git clone <repository-url>
cd MyPortfolio
open index.html  # or double-click the file to open in your browser
```

Replace `<repository-url>` with the URL of the GitHub repository.

Alternatively, you can simply download the project files and open `index.html` in a web browser.

## Usage

Once you have the project files opened in your browser, you can navigate through different sections (About, Portfolio, Skills, Contact) using the navigation links. For mobile users, the menu can be toggled using the hamburger icon.

## Features

- **Responsive Design**: Adapts to various screen sizes, providing a smooth user experience on both desktop and mobile devices.
- **Portfolio Section**: Displays a collection of projects with descriptions and images.
- **Skills Section**: Lists relevant skills in a visually appealing format.
- **Contact Information**: Easy access to contact Ari for inquiries or collaborations, including email and social media links.
- **Modern Aesthetic**: Built with a clean, modern design approach using Tailwind CSS.

## Dependencies

This project utilizes the following libraries:

- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Font Awesome**: For vector icons and social media icons.

These libraries are included via CDN links in the HTML file.

  └── index.html          # Main website file containing HTML structure
